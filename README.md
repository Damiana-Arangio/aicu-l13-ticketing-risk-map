# Materiali studenti - Lezione 13

## Cosa devi portare via

Non devi compilare un documento lungo.

Devi saper scrivere una nota minima:

```txt
Comportamento:
Rischio:
Controllo:
Comando candidato:
```

## File

- `consegna.md` - attivita' guidata.
- `template/risk-map.md` - supporto opzionale se vuoi ordinare piu' rischi.
- `template/test-candidate-list.md` - supporto opzionale per scegliere cosa scrivere in L14.

## Output atteso

Una nota breve collegata a `urgencyLabel`.

Esempio:

```txt
Comportamento: alta + telefono -> intervento rapido
Rischio: mapping cambiato senza accorgersene
Controllo: urgencyLabel deve essere "intervento rapido"
Comando candidato: pnpm test:unit
```
